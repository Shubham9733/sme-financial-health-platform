# SME Financial Health Assessment Platform

This is a **complete, runnable project** for an AI-powered financial health assessment platform for SMEs.

## Tech Stack
- Backend: Python (FastAPI, pandas)
- AI Layer: OpenAI GPT-5 (placeholder integration)
- Frontend: React.js
- Database: PostgreSQL
- Security: TLS-ready, encrypted DB
- Deployment: Docker Compose

## How to Run
1. Install Docker & Docker Compose
2. Run:
   docker-compose up --build
3. Open:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:8000/docs

No additional setup required.
